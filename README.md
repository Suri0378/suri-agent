# Suri Agent – Unified AI Companion

## What Suri Agent Does
Suri Agent is a unified AI platform that combines:
- 🔐 Cybersecurity Agent – Real-time threat monitoring, log scanning, and incident alerts.
- ☁️ Cloud Automation Agent – Manage Azure resources, deployments, and automated reporting.
- 📰 AI News Agent – Daily AI & cybersecurity updates with LinkedIn-ready posts.
- 🎙️ Voice Assistant Agent – Siri-like interface for hands-free interaction.

## How to Use It
- Visit the site: [Live Deployment] suri-agent12-hpl7hmjmo-suri379.vercel.app
- Navigate between pages using the top menu (Home, About, Contact).
- Explore the homepage for an overview, the About page for mission details, and the Contact page to reach out.
- Future versions will include real-time dashboards and backend integrations.
